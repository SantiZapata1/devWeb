Colores en CSS

herramienta para colores en HTML

Existen 4 formas diferentes de usar los colores en CSS

  • Por su nombre: Existe una lista de colores que se pueden utilizar indicando su nombre
  • Sistema hexadecimal: se basa en el sistema de numeracion hexadecimal (16 digitos), estos colores seran representaciones de 6 digitos, los 2 primeros representa la intensidad del rojo, los del medio el verde y los del final el azul, por (RGB). Agregandole 2 digitos que van de 00 a 99 indicamos la opacidad o transparencia del color
  • RGB (Red Green Blue): se usa en medios digitales, para usar este sistema de colores se utiliza una funcion que recibe 3 parametros, que son los valores de cada uno de los colores, que van de 0 a 255. Para usar la opacidad con esta funcion debemos usar rgba con valores que can del 0 al 1
  • HSL (Hue Saturation Loghtness): Este es un sistema que se basa en representar los colores indicando tono, saturacion y luminosidad. Para el tono debemos ayudarnos del circulo cromatico, en donde a traves de sus grados recorremos los diferentes tonos que van de 0 a 360 grados. El segundo es la intensidad del color que queremos y va de 0% a 100%. Por ultimo, el valor que queda es la luminosidad, este valor dara que tan luminoso o apagado deseamos que este el color

Cual usar?

Cualquier sistema de colores se puede usar, pero segun lo que utilizan los navegadores, es una buena practica si trabajamos con el sistema rgb. Aunque depende mucho de cada uno, pero lo que si es importante es usar el mismo sistema siempre.

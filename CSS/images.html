Estilos en imagenes

Dentro de los estilos que se les puede dar a las imagenes, una de las mas importantes es hacer responsive una imagen

Codigo para hacer responsive una img 

        img{
            max-width: 100%;
            height: auto;
        }

Imagenes con tamanos especificos

Objet-fit La propiedad CSS object-fit indica cómo el contenido de un elemento reemplazado, por ejemplo un img o video, debería redimensionarse para ajustarse a su contenedor. Se puede alterar la alineación del contenido del elemento reemplazado utilizando la propiedad object-position.

Propiedades de object-fit

  • fill: Es el valor por defecto, Modifica el tamaño de la imagen para llenar el cuadro de contenido. El objeto completo ocupará todo el espacio de la caja. Si el tamaño del elemento no concuerda con el de su caja, se estirará para llenarlo.
  • cover: El contenido reemplazado se dimensiona para mantener su relación de aspecto mientras llena el cuadro de contenido completo del elemento. Si la relación de aspecto del objeto no coincide con la relación de aspecto de su caja, entonces el objeto se recortará para que se ajuste.
  • contain: se muestra toda la imagen ya que se la redimensiona para que entre en su contenedor, es probable a que queden espacios en blanco
  • none: la imagen se muestra en el tamano original
  • scale-down: Toma el valor que sea menor de los valores contain y none (no recomendada)

Posicionamiento CSS

La propiedad FLOAT y POSITION nos permiten modificar la posicion natural de cualquier elemento del documento HTML. Ya que el navegador coloca cada elemento teniendo en cuenta el orden en el que aparece en el documento y segun su tipo de visualizacion (elemento de linea o bloque) Y con estas propiedades podemos cambiar esta forma de visualizacion.

Flujo normal

El Flujo Normal, o también Flow Layout, es la forma en que los elementos de línea y de bloque son presentados en una página sin que haya cambios en su diseño. El flujo es esencialmente un grupo de elementos que se perciben entre sí e interactúan entre ellos en nuestro diseño. Cuando uno de ellos se encuentra fuera del flujo, éste se comporta de manera independiente.

position

Un elemento posicionado es aquel elemento que ha salido de su flujo normal a través de la propiedad position, además adquiere nuevas propiedades. La propiedad position establece en que punto de la pagina comenzara a posicionarse, mostrarse o dibujarse el elemento que se haya establecido en el código HTML.

Tipos de posicionamiento

  • Static (default): Todos los elementos HTML son posicionados como estáticos por defecto, esto es, para hacer que el elemento fluya en el orden natural de la pagina. Si el elemento tiene este tipo de posicionamiento los parametros de top, bottom, left y rigth quedan anulados
  • relative
  • absolute
  • fixed
  • sticky

Propiedades

Para mover los elementos posicionados se activan las propiedades

  • top
  • bottom
  • left
  • right
  • z-index(cuando dos o mas elementos se solapan, podemos decidir cual aparece primero y cual por detrás de el)

Ejemplos:

caja estatica
Es el valor por defecto, significa que el elemento respeta el flujo del documento HTML, por lo que no se puede mover ni posicionar. Las propiedades mencionadas anteriormente no se aplican si esta en este estado la caja
caja relativa
El valor de posicionamiento relativo significa que el elemento conserva sus dimensiones y posicion original en el flujo del html. Y ademas es como que reserva el espacio que le indiquemos a traves de los 4 parametros
caja absoluta
El elemento pierde sus dimensiones y posicionamiento en el flujo del documento html. Por lo que hay que especificar el ancho y alto, y este siempre se ubica segun las coordenadas basandose en su contenedor padre que sea de tipo relative, si no tiene uno toma de referencia el HTML
caja fixed
el elemento pierde sus dimensiones (ancho y alto) y posicionamiento en el flujo (arriba, abajo, izq, der). La diferencia es que esta caja al realizar el scrool este elemento no se movera, quedaria fijo en el documento html, se usa para los navs en las paginas web
caja sticky Es una mezcla entre relativo y fijo, sigue el flujo de los elementos y las 4 propiedades no le indican al elemento donde posicionarse, sino que desde que valores debe cambiar de comportamiento este elemento (de relativo a fijo),a su vez debe estar contenida dentro de un elemento con dimensiones especificadas

z-index

Sirve para realizar interfacez web, como un app de diseno de templates, esta propiedad sirve para darle las diferentes profundidades a los elementos html.Determina que va adelante y que va atras. A menor valor mas al fondo estara el elemento, y a mayor mas al frente. De forma predeterminada el valor de z-index esta en auto, por lo que los elementos seguiran el orden en el que nosotros vayamos desarrollandolos (los mas viejos al fondo y los recientes encima).
Para que esta propiedad funcione la position de los elementos debe ser cualquiera menos static

implementacion: Lo ideal es usar los valores de z-index de 10 en 10, para poder trabajar facilmente si usamos varias capas en nuestros disenos

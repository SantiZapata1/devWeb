Introduccion:

CSS es un lenguaje de definicion de los estilos de los elementos html de una pagina web. Define el estilo o como se veran los distintos elementos como titulos, parrafos, secciones, imagenes, etc. La combinacion de HTML y CSS permiten crear paginas web estaticas. (solo lectura)

Atributos mas usados

  1. color: determina el color de la letra
  2. background-color: determina el fondo del selector
  3. border: determina el grosos, estilo y color del borde
  4. text-align: determina la posicion del texto
  5. font-size: determina el tamano de la letra
  6. font-family: determina la familia o el tipo de letra
  7. text-decoration:establece o elimina la decoración del texto

Selectores basicos

Un selector es la representacion de un objeto al que nosotros le podemos aplicar los estilos, existen muchos, pero los 3 basicos son:

  1. Selector de clase: Las clases generales son como plantillas con ciertos atributos especificos que se pueden asignar a distintos objetos, la ventaja de usarlas es que cada elemento puede implementar muchas clases

    Es importante el buen uso de las clases ya que su buena implementacion nos permitira reutilizar codigo y adaptarnos facilmente a diferentes frameworks. Debemos poder nombres descriptivos a las clases, que las clases sean generales y no tan especificas, para asi mediante las combinaciones de diferentes clases se pueden disenar distintos elementos

  2. Selector de id: identificadores de los elementos html # se considera una mala practica dar estilos de esta forma
  3. Selector de etiquetas: son los elementos HTML la desventaja es que todos los elementos tendran el mismo estilo

Bloque de declaraciones {}: Al margen de que selector usemos, cada uno tiene un espacio en donde declaramos el valor de cada atributo de cada selector

ejemplo: code{color:grey; background-color:black}

selectores avanzados

hijos directos

Estos selectores e usan para darle estilo a aquellos elementos que estan dentro de otro, pero son los primeros, es decir que no hay ningun elemento entre estos y su elemento padre. Si usamos este selector y el hijo no es directo no funcionara. Este selector se representa con un ">"

ejemplo:

ol > li { }
  1. primavera
  2. verano
  3. invierno

hijos descendientes

En este caso, son los elementos que estan dentro de otro pero ademas estan contenido en otro elemento, es decir que hay elementos entre ellos y su elemento padre, no son hijos directos. Se representan con un espacio en blanco

ejemplo

ol b{ }
  1. primavera aaaaa
  2. veranoaaaaaaa
  3. inviernoaaaaaa

Uso: Esto se implementa creando una clase y asignandola al elemento padre, que segun que selectores le siguen aplicara el estilo a sus elementos hijos directos o descendientes

selector de hermanos (en general)

El combinador ~ separa dos selectores y selecciona el segundo elemento sólo si está precedido por el primero y ambos comparten un padre común.

selector de hermanos adyacente

Se hace referencia a este selector como selector adyacente o selector del próximo hermano. Sólo seleccionará un elemento especificado que esté inmediatamente después de otro elemento especificado.

selectores de atributos

se usan para dar estilos a todos los elementos que tengan un atributo especifico que le indiquemos nosotros, por ejemplo en un nav que se les aplique un color a todos los elementos a que tengan el atributo target:"_blank". En general, hay diferentes formas de aplicar estilos a elementos que cumplan con ciertas caracteristicas.
Su implementacion se realiza colocando el selctor del elemento principal seguido de corchetes que en su interior tendran la condicion que deben cumplir para que se apliquen los estilos.

algunos ejemplos de selectores por atributos son:

  • Elemento a que tenga el atributo title
  • Elemento a que coincida con cierta URL
  • Elemento a que su URL contenga cierta letra o palabra
  • Elemento a que su URL comience con cierta letra o palabra
  • Elemento a con un href que termine en ".org"
  • Elementos a cuyo atributo class contenga la palabra "logo"
  • Entre otros

selector universal

Va a aplicar el estilo a todos los elementos html del documento, se designa con el simbolo "*". Tambien se la puede usar combinado con otra etiqueta para aplicar a todos los elementos que esten contenido en otro elemento

ejemplo

ol * {}
  1. primavera
  2. verano
  3. invierno

Pseudoclases

Las pseudoclases en CSS son selectores que te permiten seleccionar y estilizar un elemento en función de su estado o posición dentro del documento HTML. Estos estados pueden ser interacciones del usuario, como "hover" (cuando el mouse está sobre el elemento), "active" (cuando el usuario hace clic en el elemento) o "visited" (cuando el enlace ya ha sido visitado). También hay pseudoclases que se aplican a ciertos tipos de elementos, como "first-child" (primer hijo) o "nth-child" (ésimo hijo).
selector:pseudoclase { propiedad: valor; }

Pseudoelementos

Al igual que las Pseudo-classes, los pseudo-elementos se añaden a los selectores, pero en cambio, no describen un estado especial sino que, permiten añadir estilos a una parte concreta del documento. Por ejemplo, el pseudoelemento ::first-line selecciona solo la primera línea del elemento especificado por el selector.

El algoritmo de css

Es la forma en la que el navegador aplica los estilos CSS al documento HTML

Los podemos dividir en:

  1. La cascada

    1. User agent
    2. personalizaciones del usuario
    3. Estilos del autor

  2. La espeficidad
  3. La herencia

La cascada

Es el mecanismo en la que el navegador le va aplicando los estilos a los diferentes elementos.
Los primeros estilos que le aplican son los que estan predeterminador por el navegador que estemos usando, luego las configuracion del navegador del usuario y por ultimo el codigo css del programador. De forma secuencial los estilos css se van aplicando

La espeficidad

Viene de peso especifico, cada elemento html tiene su propio peso, y los estilos css se van a aplicar no solo en el orden que aparecen en cascada, sino que tambien dependen del peso, ganan siempre los de mayor peso especifico. Este aspecto el peso que tiene un selector y es importante cuando hay conflictos de estilos.

Usar !important se considera una mala practica porque rompe con la especificidad

La herencia

Es la capacidad de que un seelctor herede los estilos de su elemento padre, y esto se puede lograr asignandole la propiedad inherit a cada atributo que deseamos heredar.
Si deseamos evitar la herencia, esta quedaria cancelada si le asignamos otros valores a nuestro selector o si le colocamos la propiedad initial a cada atributo

Normalize

Normalize es un fichero CSS, cuyo principal objetivo es mantener los estilos similares en los navegadores, ya que cada uno agrega sus propios estilos por defecto. Por lo que esta hoja de estilo describe los estilos "estandares" que tendra cada elemento HTML.

Prefijos

Los proveedores de navegadores a veces agregan prefijos a las propiedades de CSS experimentales o no estándar y las API de JavaScript, por lo que los desarrolladores pueden experimentar con nuevas ideas mientras que, en teoría, evitan que se confíe en sus experimentos y luego rompan el código de los desarrolladores web durante el proceso de estandarización. Los desarrolladores deben esperar para incluir la propiedad sin prefijar hasta que se estandarice el comportamiento del navegador.

Los principales navegadores usan los siguientes prefijos:

  1. -webkit- (Chrome, Safari, nuevas versiones de Opera, casi todos los navegadores iOS (incluyendo Firefox for iOS); basicamente, cualquier navegador basado en WebKit)
  2. -moz- (Firefox)
  3. -o- (Versiónes antiguas, pre-Webkit, de Opera)
  4. -ms- (Internet Explorer y Microsoft Edge)

Flexbox o flex

Concepto: Flexbox es un modelo de diseño en CSS que permite crear diseños más complejos y flexibles de una manera más sencilla que los modelos de diseño tradicionales. Con Flexbox, puedes controlar la distribución de elementos en un contenedor y alinearlos de manera fácil y rápida.

  • Contenedor flex (Flex container): Es el elemento padre que contiene a los elementos hijos (items). Para convertir un contenedor en un contenedor flex, se utiliza la propiedad `display` con el valor `flex` o `inline-flex`. Este se comporta como un elemento block o inline block.

  • Eje principal (Main axis) y eje secundario (Cross axis): En un contenedor flex, los elementos pueden ser dispuestos a lo largo del eje principal (main axis) o del eje secundario (cross axis), dependiendo de la dirección de la fila o columna. Por defecto, el eje principal es horizontal (de izquierda a derecha) y el eje secundario es vertical (de arriba hacia abajo)

  • Dirección de la fila o columna (Flex-direction): Especifica la dirección en la que los elementos se colocan en el contenedor flex. Puede ser:
    • row (de izquierda a derecha)
    • row-reverse (de derecha a izquierda)
    • column (de arriba hacia abajo)
    • column-reverse (de abajo hacia arriba)

  • Alineación de los elementos (Justify-content y align-items):
    • justify-content alinea los elementos a lo largo del eje principal
    • align-items alinea los elementos a lo largo del eje secundario.
    Los valores comunes son flex-start, flex-end, center , space-between y space-around.

  • Orden de los elementos (Order): La propiedad order permite cambiar el orden en el que los elementos son mostrados dentro del contenedor flex. Por defecto, todos los elementos tienen un valor de 0, y se pueden utilizar valores negativos o positivos para cambiar su posición.

  • Espaciado entre los elementos (Flex gap): La propiedad gap permite definir un espacio entre los elementos en un contenedor flex. Es una forma más moderna de lograr el espaciado que utilizar márgenes.

  • Flex-wrap: indicamos si permitimos o no que cuando no haya espacio para los items hijos se bajen a una nueva linea, esta propiedad se la da al padre. Gracias a esto los elementos no se van a encoger sino que van a bajar de linea.

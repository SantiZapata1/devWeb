Modelo de cajas CSS

Para HTML todos los elementos son cajas que tienen relleno (padding), borde (border ) y margen (margin).

Un elemento puede ser de bloque (ocupa todo el espacio) o de linea (ocupa lo minimo e indispensable)

  • CONTENT: es el contenido del elemento HTML, tomando en cuenta sus dimenciones (width y height)
  • padding: distancia interna de la caja (del contenido al borde)
  • border: borde de la caja, es lo que delimita cada elemento HTML
  • margin: distancia externa de la caja (del borde hasta el proximo elemento)

Aclaracion: Dentro del modelo de cajas hay que considerar que una caja tiene 4 lados

  1. Arriba (top)
  2. Derecha (right)
  3. Abajo (bottom)
  4. Izquierda (left)

elementos en bloque y en linea

Cajas en linea

  • Ocupan solo el espacio necesario
  • No tienen dimenciones modificables (ancho y alto)
  • Gracias a esto permite que otros elementos esten a su lado
  • Las propiedades padding y margin solo se aplicaran de forma horizontal, verticalmente los ignora. En la practica el elemento toma verticalmente tambien pero pierde el flujo.

Cajas en bloque

  • Ocupan todo el ancho disponible
  • Tienen dimenciones modificables (ancho y alto)
  • No permiten que otros elementos esten a su lado (por mas que su contenido no use todo el ancho), por lo que provocan saltos de linea

Propiedad display

Esta propiedad le indica al navegador como se debe comportar el elemento html, ademas de los atributos inline y block esta otros que son flex y grid, que son modulos de maquetacion que se usan para maquetar paginas web de diferentes maneras, cada una tiene su propia implementacion.

Distintos atributos de Display:

  • inline: el elemento se comportara en linea
  • block: el elemento se comportara en bloque
  • inline-block: el elemento se comportara como en linea pero aceptara modificaciones de ancho y alto como uno en bloque
  • none: carga pero oculta el elemento html
  • flex: modelo de maquetacion
  • grid: modelo de maquetacion

Tamano de caja o box-sizing

Esta propiedad define como el alto y ancho de una caja es calculado, el valor predeterminado es content-box, y este toma en cuenta el ancho y alto que le pasemos para darselo al CONTENIDO y los valores de padding, borde y margin se agregan a eso. Sin embargo si usamos border-box las medidas que le pasemos se le restaran al padding, borde y margin que tengamos, por lo que nuestro tamano de contenido se ira achicando

Existe una tecnica para no estar aplicandole a todos nuestros elementos esta propiedad y es usando este codigo css:

html{
box-sizing:border-box
}

*,*:before,*:after{
box-sizing:inherit
}

Escribiendo este codigo css al inicio del programa reseteas todos los tamanos de los elementos para no tener que estar realizando calculos y arreglos tediosos todo el tiempo a nuestros elementos html

Float y clear

float La propiedad CSS float ubica un elemento al lado izquierdo o derecho de su contenedor, permitiendo a los elementos de texto y en línea aparecer a su costado. El elemento es removido del normal flujo de la página, aunque aún sigue siendo parte del flujo (a diferencia del posicionamiento absoluto).Como se ha mencionado más arriba, cuando un elemento flota, es removido del flujo normal del documento (aunque sigue perteneciendo a él). Se cambia hacia la izquierda, o hacia la derecha, hasta que toca el límite de su caja contenedora, u otro elemento flotante.
Esto ya no se utiliza por los modelos de maquetacion, como grid o flex box. Igual es importante entender el concepto porque paginas web viejas pueden implementar estas propiedades.

Limpiando (clearing) flotantes A veces querrás forzar un item a moverse por debajo de elementos flotantes. Por ejemplo, párrafos que han de permanecer adyacentes a elementos flotantes, pero forzar a los encabezados a estar en su propia línea. Para ello revisa el siguiente ejemplo: clear

Colapso de margenes

Los márgenes Top (en-US) y bottom de los bloques a veces están combinados (colapsados) en un solo margen cuyo tamaño es el mayor de los márgenes combinados, un comportamiento conocido como colapso de margen. Ten en cuenta que los márgenes de flotantes y elementos con posición absoluta nunca colapsan.

Para centrar las cajas al tamano de las diferentes pantallas de lleva a cabo aplicando a la caja contenedora la propiedad margin-left: auto; margin-right: auto;

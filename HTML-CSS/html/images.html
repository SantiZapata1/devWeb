Estilos en imagenes

Los estilos CSS que puedes aplicar a una imagen son diversos y te permiten controlar cómo se ve y se comporta la imagen en tu página web.

Imagenes responsivas

Dentro de los estilos que se les puede dar a las imagenes, una de las mas importantes es hacer responsiva una imagen

        img{
            max-width: 100%;
            height: auto;
        }

Imagenes con tamanos especificos

Cuando necesitamos que el elemento "imagen" tenga un tamano en especifico en nuestra pagina web, se la podemos dar con valores fijos o dinamicos, pero tambien podemos configurar como va a comportarse el contenido de la imagen en ese espacio que le indiquemos

Objet-fit

En HTML, cuando colocas una imagen en tu página web, puedes controlar cómo se muestra en su contenedor utilizando la propiedad object-fit en CSS. Esta propiedad te permite especificar cómo la imagen debe ajustarse al contenedor.

Objet-position

La propiedad object-position en CSS se utiliza para especificar la posición inicial de una imagen dentro de su contenedor. Esta propiedad toma dos valores: uno para la posición horizontal y otro para la posición vertical.

Propiedades de object-fit

  • fill: valor por defecto La imagen se ajustará para llenar completamente el contenedor, sin mantener su relación de aspecto.
    Esto puede hacer que la imagen se vea distorsionada si la relación de aspecto del contenedor y la imagen no coinciden. Prioriza que se vea toda la imagen

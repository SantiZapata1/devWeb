Fondos en css

Como en muchas propiedades, para usar las que son para los fondos tambien hay propiedades especificas y shorthands para ahorrar codigo, pero por temas de renderizacion y demas es mejor usar las propiedades especificas antes que los shorthands

Propiedades para el background o fondo:

  1. Background-color
  2. backgorund-image
  3. background-size manual, contain y cover
  4. background-repeat repeat y no-repeat
  5. background-position
  6. background-clip
  7. background-origin
  8. background-attachment

  • background-color Define el color de fondo del elemento

  • background-image Define la imagen de fondo del elemento, en complemento con la funcion url() donde esta la direccion de la imagen. Si no se especifica un tamano, estara en su tamano original.

  • background-size Define el tamano de la imagen de fondo Se lo puede hacer de diferentes maneras:

    Manualmente: la primera es indicando el tamano de ancho y alto para que la imagen se adapte a eso. Lo mas probable es que la imagen se deforme, ya que su tamano original es otro al que le indiquemos, pero este se va a estirar o acordar para cumplir con lo que le indiquemos.

  • background-size Define el tamano de la imagen de fondo Se lo puede hacer de diferentes maneras:

    Cover: La segunda es colocandole el valor de "cover" que lo que hace es priorizar que todo el elemento tenga la imagen, por lo que si es necesario se va a modificar el tamano real de la imagen para que el elemento este relleno. no estira pero si escala y recorta

  • background-size Define el tamano de la imagen de fondo Se lo puede hacer de diferentes maneras:

    contain: Y el tercer valor es "Contain", que cambia el tamano de la imagen si es necesario pero se vera en sus dimensiones originales, y si falta espacio por rellenar se repetira hasta que este cubierto. no recorta ni estira, solo escala

  • background-repeat Define como se repite la imagen de fondo del elemento, si falta rellenar espacio la repetira a la imagen (de diferentes maneras), pero con la propiedad no-repear no repetira la imagen por mas que no rellene todo el elemento

  • background-repeat

  • background-position Define posicion en "x" e "y" que deseamos que este la imagen de fondo. Tambien podemos indicar que la posicion sea bottom, left, top, right, o combinaciones de estos.

  • background-clip (fondo) Determina desde cuando arranca a reproducirse el color de fondo, si desde el borde, padding o content

  • background-origin (imagen) Determina desde cuando arranca a reproducirse la imagen de fondo, si desde el borde, padding o content

  • background-attachment Define como se comporta la imagen de fondo ante el scroll de la pagina
